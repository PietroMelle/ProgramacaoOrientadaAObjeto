▪ Vamos fazer uma classe de carros, que tenha um método que ao ser acionado pelo objeto possa calcular quantos anos o modelo de carro tem em relação ao ano atual.

▪ Vamos construir uma classe com os seguintes atributos:
1. Marca
2. Modelo
3. Cor
4. Ano de Fabricação

▪ O método usará uma função do JS (date) que pode nos fornecer o ano atual.

▪ Exemplo de uso:
const dia = new Date();
const ano = dia.getFullYear();
document.getElementById("tempoDeUso").innerHTML = ano;
